# Login Form (Login-Form-24)

Login Form built using HTML, CSS and JavaScript with a color-changing background.

![Login Form Screenshot](./Screenshot%202024-06-08%20210550.png)

## Overview

A simple, responsive login form that demonstrates clean HTML structure, modern CSS styling, and a small JavaScript component that animates the page background colors. This repository is ideal for beginners who want a compact example of a polished form UI and simple front-end interactivity.

## Features

- Elegant, centered login form layout
- Smooth color-changing background animation
- Basic client-side validation (JavaScript)
- Responsive and lightweight (no build tools required)

## Files

- `index.html` — Markup for the login form
- `style.css` — Styles and layout, including background animation
- `script.js` — JavaScript for background color transitions and simple validation
- Screenshot images demonstrating the form appearance

## Usage

To run the project locally:

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Login-Form-24.git
2. Open `index.html` in your browser (double-click or use `file:///` path).

No server or dependencies required — everything runs in the browser.

## Customization

- Change colors or animation timings in `style.css`.
- Update validation rules or add form handling in `script.js`.
- Replace screenshot images with your own capture if you update the UI.
